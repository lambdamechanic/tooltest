# MCP schema resources

These JSON schema files are copied from the MCP specification repo:
https://github.com/modelcontextprotocol/specification/tree/main/schema

Each schema has a companion `.source.txt` file recording the upstream ref and
SHA-256 checksum for reproducibility.

To refresh:

```
./scripts/update-mcp-schema.sh 2025-11-25 <git-ref>
```
